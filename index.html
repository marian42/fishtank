About

Hardware

This is my internet-connected fishtank. It's powered by a Raspberry Pi. It features a fish feeder, a camera and a light switch. The fish feeder is an automatic fish feeder that was modified to empty an arbitrary container at any time, instead of only the next container each day. Inside the fish feeder there is an Arduino Pro Mini, some sensors and a motor (read more). The Raspberry Pi can switch the light with an Elro remote power switch by sending a command through a 433Mhz transmitter. Finally, there is a webcam to observe the fish.

Software

The project uses a Flask / Python / Apache server and Javascript and Bootstrap on the client side.

Find the source code on github.

Events

The Raspberry Pi will perform scheduled events like feeding, switching the light and taking pictures. It skips feed events if the fish are not hungry.

To determine the hungriness of the fish, a value is stored and increased each time the fish are fed. It gradually decreases by 1 each day but it doesn't go below 0. If the saturation value is below 1 and the lights are on, any guest may click a container and feed the fish.

The server uses the Pushbullet API to send me a picture of my happy fish right after they have been fed. It also notifies me if the food containers need to be refilled.
